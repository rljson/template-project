# Install WSL and Ubuntu

## Install WSL

Press `Windows`

Type `wsl` and press `Enter`

Press a button to install WSL

## Install Ubuntu

Press `Windows`

Type `powershell` and press `Enter`

Get a list of Ubuntu installations:

```bash
wsl --list --online
```

```bash
wsl --install Ubuntu-22.04
```

Enter username `dsdev`

Enter and remember password

## Start Ubuntu

Press `Windows`

Type `wsl` and press `Enter`

Ubuntu image is started automatically
